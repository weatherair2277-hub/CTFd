# overview
this repo contains a plugin for the ctfd platform which reveals the current flags of solved challenges. if you don't remember the challenge's flag, or if the challenge's flag has changed but your submission is still considered valid, this plugin will help you.

# use
to use the plugin, simply put the repo inside the plugins directory

(e.g. from github `git clone "https://github.com/IdanRosenzweig/ctfd_plugin_reveal_flag" "_parent_dir_/CTFd/CTFd/plugins/ctfd_plugin_reveal_flag"`)
