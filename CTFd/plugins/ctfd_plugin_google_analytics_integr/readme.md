# overview
this repo contains a google analytics integration plugin for the ctfd platform. it provides an simple and easy interface for connecting google analytics to the site.

# use
to use the plugin, simply put the repo inside the plugins directory

(e.g. from github `git clone "https://github.com/IdanRosenzweig/ctfd_plugin_google_analytics_integr" "_parent_dir_/CTFd/CTFd/plugins/ctfd_plugin_google_analytics_integr"`)
