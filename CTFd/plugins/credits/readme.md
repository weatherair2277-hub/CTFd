# overview
this repo contains a credits plugin for the ctfd platform. you can use credits to acknowledge users and teams for finding unintended solutions, suggesting improvements, helping the community and more.

# use
to use the plugin, simply put the repo inside the plugins directory

(e.g. from github `git clone "https://github.com/IdanRosenzweig/ctfd_plugin_credits" "_parent_dir_/CTFd/CTFd/plugins/ctfd_plugin_credits"`)
