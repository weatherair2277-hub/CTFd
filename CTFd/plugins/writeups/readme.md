# overview
this repo contains writeups plugin for the ctfd platform. the plugin allows users to upload writeups for challenges they solved and view other users' writeups.

# use
to use the plugin, simply put the repo inside the plugins directory

(e.g. from github `git clone "https://github.com/IdanRosenzweig/ctfd_plugin_writeups" "_parent_dir_/CTFd/CTFd/plugins/ctfd_plugin_writeups"`)
