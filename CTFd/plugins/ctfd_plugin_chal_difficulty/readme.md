# overview
this repo contains a chal difficulty plugin for the ctfd platform. you can set the difficulty for each challenge.

# use
to use the plugin, simply put the repo inside the plugins directory

(e.g. from github `git clone "https://github.com/IdanRosenzweig/ctfd_plugin_chal_difficulties" "_parent_dir_/CTFd/CTFd/plugins/ctfd_plugin_chal_difficulties"`)
